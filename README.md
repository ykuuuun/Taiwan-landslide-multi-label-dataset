# Taiwan-landslide-multi-label-dataset
 A Taiwan landslide multi-label dataset was compiled, integrating Sentinel-1 dual polarization data, GF-3 full polarization data, and corresponding GF-6 optical images for the region.
 The dataset will be released after the paper is accepted.

Bare soil	1995
